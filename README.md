# 💼 HR Analytics Dashboard in Excel – Workforce Insights Project

📊 **Project Overview**  
This project focuses on turning **raw HR data** into **actionable insights** using **Microsoft Excel**.  
The dashboard provides a clear view of **employee performance, attendance, and attrition**, helping HR teams make data-driven workforce decisions quickly and effectively.  

The analysis includes:
- Data cleaning and preparation  
- Pivot Tables for summarization  
- KPI cards for key HR metrics  
- Interactive charts and slicers for dynamic filtering  
- Highlighting trends in performance, salary, and attrition  

---

## 🧰 Tools & Features Used
- **Microsoft Excel**  
- **Pivot Tables & Charts**  
- **Slicers for Interactive Filtering**  
- **Conditional Formatting**  
- **Formulas:** `IF`, `AVERAGEIF`, `COUNTIF`, `VLOOKUP`, `SUMIFS`  

---

## 🧾 Dataset Overview
The dataset includes key HR parameters such as:
- Employee ID  
- Department  
- Gender  
- Age  
- Experience (Years)  
- Attendance (%)  
- Performance Rating  
- Salary  
- Status (Active / Resigned)  

The goal is to summarize and visualize employee data to uncover trends in performance, retention, and workforce distribution.

---

📊 Final Dashboard

 My Excel dashboard includes:

 ✅ Department-wise Performance

 ✅ Attendance department-wise

 ✅ Attrition by department

 ✅ Status of employees

# 💰 Financial Tracker Dashboard in Excel

## 📘 Project Overview
The **Financial Tracker Dashboard** is an Excel-based analytics project designed to monitor **income, expenses, and profit trends** while comparing **budgeted vs. actual performance**.  
This project demonstrates how simple Excel tools can transform raw financial data into **interactive insights and visual dashboards**.

---

## 🎯 Objective
To build a **comprehensive Financial Dashboard** that helps analyze:
- Income and Expense trends  
- Category-wise spending patterns  
- Budget vs. Actual performance  
- Profit and savings percentage  

---

## 🧩 Dataset Details
The dataset includes transaction-level data with the following columns:

| Column Name | Description |
|--------------|-------------|
| Date | Transaction date |
| Category | Expense/Income category (e.g., Rent, Salary, Groceries) |
| Description | Details of the transaction |
| Type | Income or Expense |
| Amount (Actual) | Actual amount spent or earned |
| Budget | Budgeted value for that category |

---

## 🧮 Excel Techniques Used
- **Data Cleaning**: Removing blanks, correcting formats, and ensuring consistent category names.  
- **Formulas Used**:
  - `SUMIF()` – for total income and expense calculations  
  - `IF()` – for condition-based calculations  
  - `Variance` and `Variance %` formulas to compare budget vs. actual  
- **Pivot Tables** for summarizing data by category and type.  
- **Charts Used**:
  - Bar Chart → Category vs. Expenses  
  - Line Chart → Budget vs. Actual  
  - Donut Chart → Category vs. Budget  
- **Slicers** for dynamic filtering (Category, Type, Budget).

---

## 📊 Dashboard Highlights
The dashboard provides:
- **Category vs. Expenses (Bar Chart):** Identifies high-spending categories like Rent and Groceries.  
- **Budget vs. Actual (Line Chart):** Tracks budget performance over time.  
- **Description vs. Expenses (Bar Chart):** Details expense breakdown by description.  
- **Category vs. Budget (Donut Chart):** Displays how total budget is distributed.  
- **Interactive Filters:** Allow analysis by Category, Type, and Budget.  

---

## 💡 Key Insights
- Rent accounted for the highest expense (≈39% of total spending).  
- Groceries (13%) and Transport (7%) were the next major cost areas.  
- Electricity, Health, and Dining Out were within budget limits.  
- Overall savings rate: **51%**, showing efficient financial management.

- Dashboard Design and KPI Visualization  
- Budget Tracking and Performance Monitoring
